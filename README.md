# Chuck Norris API (chuck-norris)

Free REST API providing random Chuck Norris jokes, categories, and the ability to search the joke database with no authentication required. The service is hosted on AWS and supported by Jugendstil.io sponsorship, making it permanently free for developers. Responses are returned as JSON and include a joke identifier, URL, icon, and the joke text. Integrations are available for Slack and Facebook Messenger.

## APIs

### Chuck Norris API

Retrieve random Chuck Norris jokes, list available joke categories, and perform free text searches across the curated joke database. No API key or authentication is required.

- **Human URL:** [https://api.chucknorris.io](https://api.chucknorris.io)
